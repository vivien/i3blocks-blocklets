# keyboard-layout
Shows the currently active keyboard layout.

![](keyboard-layout.png)
![](keyboard-layout-fa.png)

# Dependencies

* [xkblayout-state](https://github.com/nonpop/xkblayout-state)
* FontAwesome (optional) for label

# Config
```
[keyboard-layout]
command=$SCRIPT_DIR/keyboard-layout
```
